About This Site

The primary purpose of this website is to serve as a portfolio of my personal projects I work on in my free time. Like other engineers (see "other people" in sidebar), I believe that documentation is extremely important in STEM and is often overlooked by many builders. This website will act somewhat as a personal "brain-dump" and also hopefully aid others working on similar projects. I plan to keep this site dynamic, so content is likely to be added in a semi-regular fashion.


Each project has its own dedicated project page that will be updated as progress is made. At the end of some of the project pages, there is an "extras" section that goes more indepth - normally pertaining to the math involved.
